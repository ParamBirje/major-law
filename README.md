## About the Project

A.I powered legal assistant that provides a centralized, easily accessible knowledge base for all your legal paperwork and public sources by referencing webpage URLs in the chat.

## Tech Stack

Built using,

- React.js / Vite
- Typescript
- Tailwind CSS
- Python
- FastAPI
- AWS DynamoDB

### Versions

Python 3.12.0
Node v20.11.1
